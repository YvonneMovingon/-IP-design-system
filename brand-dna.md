# Brand DNA — 特洛马克的品牌基因

> 所有场景、所有页面共享的底层规范。无论哪种设计风格，都不能违背此文件的核心约束。

## 🎨 IP固定三色

| 色名 | 色值 | 对应 | 用途 |
|------|------|------|------|
| 玫粉 | `#F25187` | 头发 | 品牌主色、标题强调、超链接、CTA |
| 墨黑 | `#403F44` | 贝雷帽/皮衣 | 正文色、版面骨架、边框、分割线 |
| 裸粉 | `#EAAAB0` | 耳环/腮红 | 柔和过渡、次要背景、装饰色、hover态 |

三色比例原则：墨黑50% · 玫粉35% · 裸粉15%（墨黑撑骨架，玫粉点品牌，裸粉做呼吸）

## 👩‍🎨 IP形象规范

特洛马克的固定视觉形象，以下元素不可更改：

- **粉色短发** — 品牌识别第一要素，微卷bob头，粉色饱和度高
- **黑色贝雷帽** — 白色圆点装饰，法式文艺感
- **黑色大圆框眼镜** — 知识分子气质+潮流感，比例偏大
- **青色S形耳环 + 红色珠坠** — 唯一的冷色点缀，制造视觉意外
- **黑色皮衣/夹克** — 酷、有态度、不乖巧
- **白色/粉色条纹衬衫** — 皮衣内搭，甜酷反差
- **红色挂绳 + 工作证** — "在场感"标志，暗示创意行业从业者
- **粉色腮红** — 温暖感，不冷漠
- **格纹/碎花混搭下装** — 拼贴感的延伸，不追求统一

### 素材文件
- **头像** — `assets/avatar.jpg`（粉色短发+贝雷帽+圆框眼镜，适用于个人介绍区）
- **表情包系列** — `char-small/char1~4.png`（四种情绪：叉腰倔强 / 握拳兴奋 / 托腮沉思 / 惊讶推眼镜）
- **全身形象** — `character.png`（完整IP形象）

### 使用规则
- 不允许修改形象比例、颜色、元素
- 是否在页面中使用IP形象由用户决定，不强制
- 需要头像时优先用 `assets/avatar.jpg`

## 🔤 字体基因

### 核心原则
- **标题用衬线，正文用无衬线** — 编辑感的节奏，像杂志版面
- **中英文搭配** — 英文做装饰/标签，中文承载内容
- **字号对比极端** — 大的要很大，小的要真的小
- **允许等宽体做标签** — 拼贴感的一部分，像打印机标签

### 推荐字体池

| 场景 | 推荐 | 备注 |
|------|------|------|
| 英文装饰/标题 | `Playfair Display` (italic) | 杂志编辑感衬线体，优雅有锋利度 |
| 英文标签/数据 | `Space Mono` | 有个性的等宽体，适合拼贴标签、编号 |
| 英文手写/注释 | `Caveat` | 手写标注感，随意但不潦草 |
| 中文标题首选 | `Noto Serif SC` (900) | 高粗度衬线体，编辑感 |
| 中文标题备选 | `LXGW WenKai`（霞鹜文楷） | 手写楷体，有温度但不甜腻 |
| 中文正文 | `Noto Sans SC` + 系统栈 | 跨平台无衬线 |
| 中文正文系统栈 | `-apple-system, 'PingFang SC', 'Helvetica Neue', sans-serif` | 本地渲染最快 |

### 字号系统（fluid sizing）
- Hero大标题: `clamp(2.8rem, 7vw, 5.5rem)`
- Section标题: `clamp(1.6rem, 4vw, 2.6rem)`
- 卡片标题: `1.15rem ~ 1.4rem`
- 正文: `16px`
- 辅助文字/标签: `0.75rem ~ 0.85rem` + `letter-spacing: 0.08em`（标签要加字距）
- 大装饰数字: `clamp(3rem, 8vw, 7rem)` + `opacity: 0.08~0.15`

## ✨ 气质关键词

设计出来的东西应该让人觉得：

- **杂志编辑感** — 像翻开一本有态度的独立杂志，不是公众号排版
- **拼贴剪报风** — 不规则排版、叠加元素、撕裂边缘、手工贴纸质感
- **Riso印刷质感** — 微颗粒、半透明叠色、纸张肌理、不完美的美
- **粉黑碰撞** — 甜和酷共存，不走极端，不选边站
- **酷但不冷** — 有温度的锋利，有观点但不刻薄
- **不像AI** — 最高优先级约束。宁可"不够精致"也不要"一眼AI"
- **有态度的可爱** — 可以有少女心但骨子里是一个有主见、不讨好的人

### 一句话总结
> 如果不二是"手绘蜡笔感的可爱知识女孩"，特洛马克就是"拼贴杂志感的甜酷创意女生"。

## 🎨 配色扩展原则

当三色不够用时：

### 背景色系
- 亮场主背景：`#FCF8DF`（奶白）或 `#FFF5F5`（微粉白）
- 亮场次背景：`#FECBCA`（浅粉底）——卡片底、引用块底
- 暗场主背景：`#1A1A1A`（近墨黑）或 `#0D0D0D`（深夜黑）——允许用于全屏页面和3:4卡片
- 暗场次背景：`#2A2A2A`（暖灰黑）——卡片、弹窗

### 文字色系
- 主文字永远非纯黑：用 `#1A1A2E`（墨色）或 `#403F44`（品牌墨黑）
- 次要文字：`#666`、`#888`
- 亮底上的辅助色：`#999`
- 暗底上的主文字：`#F5F0EB`（暖白），不用纯白 `#fff`
- 暗底上的次文字：`#AAAAAA`

### 功能性扩展色
- 强调黄（谨慎使用）：`#F4D758`——仅用于高亮标记、告警badge，不做大面积
- 青绿（来自耳环）：`#4ECDC4`——仅用于特殊状态、成功提示、偶尔的视觉意外
- 绝不用纯黑 `#000` 或纯白 `#fff`

### 配色模式
本品牌支持两种底色模式（不二只有亮场），根据场景选择：
- **亮场（默认）**：奶白/微粉底 + 墨黑文字 + 玫粉强调 — 温暖、亲和
- **暗场**：深黑底 + 暖白文字 + 玫粉强调 — 酷、杂志大片感

## 🚫 通用禁忌清单

| 类型 | 禁止 |
|------|------|
| 配色 | 蓝紫渐变、cyan/neon色、全粉Barbiecore（粉要和黑碰撞不要自己玩）、AI常用的冷灰蓝调 |
| 字体 | Inter/Roboto/Arial等overused字体、monospace当"技术感"装饰（除非是Space Mono做标签） |
| 布局 | 所有section居中对称、千篇一律卡片网格、cards嵌套cards、纯列表罗列 |
| 动效 | bounce/elastic、animate width/height、无限循环动画 |
| 装饰 | glassmorphism、渐变文字、AI光效、圆角矩形+阴影千篇一律 |
| 整体 | 看起来像AI生成的通用模板、generic Landing Page感、公众号排版风 |
| 粉色使用 | 全页面粉色、粉色渐变做背景、粉+紫的"少女心"配色——粉色永远要有墨黑制衡 |
| 排版 | 行间距/字间距过松或过紧、所有文字居中、缺少层次对比 |
| 图片 | AI生成的stock photo风、过度滤镜、无意义装饰图 |
| 默认样式 | HTML默认blockquote、默认border-left引用块、无样式ul/ol列表、默认table——所有组件必须从components.md选用 |

### 粉色使用的特别注意
粉色是品牌的灵魂，但滥用会变成"又一个粉色少女风"。规则：
1. **玫粉(#F25187)永远搭配墨黑(#403F44)出现**——单独大面积使用禁止
2. **裸粉(#EAAAB0)可以做大面积背景**——但必须有墨黑文字/元素压住
3. **不允许粉色渐变**——平色或半透明叠色（Riso风格），不做smooth gradient

### 自检问题
做完设计后问自己：
1. 这个页面截图发到小红书上，会不会被人评论"又是AI做的"？
2. 能不能一眼认出这是"特洛马克的"？——有没有粉黑碰撞+杂志拼贴感？
3. 有没有哪个部分让你觉得"在Canva模板里见过"？
4. 粉色有没有被墨黑制衡？还是甜过头了？

## 📐 通用间距原则

- Section之间: `clamp(80px, 12vh, 160px)`
- 内容块之间: `clamp(40px, 6vw, 100px)`
- 卡片内padding: `clamp(28px, 3vw, 44px)`
- 元素间gap: `clamp(24px, 3vw, 48px)`
- 全部用 `clamp()` 做fluid sizing
- `max-width: 1300px` + `margin: 0 auto` 约束内容宽度

## 📱 响应式通用规则

- 断点: 900px（两栏→单栏）、600px（字号微缩）
- 移动端是"重新排列"不是"缩小"
- 尊重 `prefers-reduced-motion`
- 移动端不隐藏内容——adapt不amputate

## 🖨️ 拼贴/Riso质感实现指南

本品牌的视觉差异化核心是"拼贴杂志+Riso印刷"质感，以下是实现规范：

### Riso纹理
- 使用CSS `background-image` 叠加噪点纹理：`url(data:image/svg+xml,...)` 或微小的 noise pattern
- 颗粒度：细颗粒（grain），不要粗噪点
- 透明度控制在 `opacity: 0.03~0.08`，隐约可见即可
- 色彩叠加：两个半透明色块重叠时允许 `mix-blend-mode: multiply`

### 拼贴元素
- **撕裂边缘**：用不规则的 `clip-path` 或 SVG mask 模拟纸张撕裂
- **贴纸标签**：小段文字用 `transform: rotate(-2deg~3deg)` 微微歪斜，像手贴上去的
- **胶带装饰**：半透明条状元素，`opacity: 0.3~0.5`，横跨在图片或卡片边缘
- **手绘圈注**：用 SVG 的不规则椭圆做圈注强调，线条要抖（不能是完美的圆）

### 版面打破
- 允许元素溢出容器边界（`overflow: visible` + `translateX/Y`）
- 图片可以不对齐，轻微旋转 `rotate(-1deg~2deg)`
- 文字和图片可以重叠，用 `z-index` 控制层次
- 不追求像素级对齐——"差一点"才对

## 🔍 细节规范

- **选中文本高亮**: `::selection { background: #F25187; color: #fff; }`
- **链接默认**: 玫粉色 `#F25187`，无下划线
- **链接悬停**: 裸粉色底色块 `background: #EAAAB0`，或加下划线
- **按钮主色**: 玫粉底+白字，hover态变深 `#D43D6F`
- **按钮次要**: 墨黑底+暖白字，hover态玫粉描边
- **边框装饰**: 如需装饰性边框，用墨黑 `#403F44`，粗度 `2px`（细线编辑感），不用粗框
- **分割线**: `border-top: 1px solid #EAAAB0`（裸粉细线）或 `#ddd`
- **卡片阴影**: 不用 `box-shadow` 做立体感——用实色偏移阴影 `box-shadow: 4px 4px 0 #403F44`（Riso/版画风格）

## 🗣️ 品牌语气与设计的关系

特洛马克的写作风格是"观点先行、短句推进、智识与感性并存"，设计语言要匹配：

- **不对称 > 居中** — 观点不是两边讨好的，版面也不该是
- **留白要狠** — 短句的呼吸感，对应版面的大面积留白
- **对比要强** — 字号、颜色、粗细的对比要极端，像语气的轻重缓急
- **不做不痛不痒的装饰** — 每个视觉元素都有存在的理由，和写作一样
